# helpdeskeddy
1. При использовании автоматизации цикл "строгий" если попался не корректный тикет, цикл преостанавливает работу для дальшейшего исправления тикета оператором
2. При использовании автоматизации цикл "давалка" заплоняет гугл форму каждого тикета и в случае если он не корректный переходит к следующему
3. Цикл "Девственница" сразу проверяет не корректный тикет и если он не корректный не заполняет его в гугл форму
